## Provenance Filters

The [`filters.provenance`](/features/api-reference#searchfilters) parameter allows you to narrow search results within a specific connector (e.g., search within a specific Slack channel or GitHub repository).

<Warning>
**Provenance Filtering Requirements**

To use provenance filtering, you must:

- Specify **exactly one source** in `filters.sources`
- Use a source that supports provenance (see table below)

Attempting to use provenance with multiple sources or an unsupported source will result in an error.

</Warning>

Supported sources and provenance value formats:

| Source                              | Provenance Value Format        |
| ----------------------------------- | ------------------------------ |
| [`SLACK`](/connectors/slack)        | Channel name **OR** channel ID |
| [`GITHUB_PRS`](/connectors/github)  | Repository name                |
| [`GITHUB_CODE`](/connectors/github) | Repository name                |
| [`LINEAR`](/connectors/linear)      | Team name                      |
