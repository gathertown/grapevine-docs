Get up and running with Grapevine to search your company's knowledge base.

## Create an account

1. Go to [getgrapevine.ai](https://getgrapevine.ai)
2. Sign up with your work email
3. Create your organization

## Connect your first data source

<Steps>
  <Step title="Choose a connector">
    Navigate to the Integrations page and select a data source to connect.
    
    Popular choices:
    - **Slack** - Index team conversations
    - **Notion** - Index documentation
    - **GitHub** - Index code and issues
  </Step>
  
  <Step title="Authorize the connection">
    Follow the OAuth flow to authorize Grapevine to access your data source.
    
    You'll need admin permissions for most integrations.
  </Step>
  
  <Step title="Wait for indexing">
    Grapevine will begin indexing your content. This may take a few minutes to several hours depending on the amount of data.
    
    You can monitor progress on the Integrations page.
  </Step>
</Steps>

## Make your first query

Once indexing is complete, you can start searching:

### Via the web interface

1. Go to the Search page
2. Enter a natural language question
3. View results with citations

### Via API

```bash
curl -X POST https://api.getgrapevine.ai/v1/ask \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "What is our pricing strategy?"
  }'
```

## Get your API key

To use the API:

1. Navigate to Settings > API Keys
2. Click "Create API Key"
3. Copy your key (it won't be shown again)
4. Use it in the `Authorization` header
