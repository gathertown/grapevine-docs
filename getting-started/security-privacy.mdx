title: Security & Privacy
description: Learn how Grapevine protects your data and respects privacy
---

Grapevine is built with security and privacy as core principles.

## Data security

### Encryption

- All data is encrypted in transit using TLS 1.3
- Data at rest is encrypted using industry-standard encryption
- API keys and credentials are stored securely using AWS Secrets Manager

### Access control

- OAuth-based authentication for all integrations
- Permissions are inherited from source systems
- Users only see content they have access to in the original source

## Privacy

### Data handling

- Grapevine only indexes content you explicitly connect
- You maintain full control over what data is indexed
- Data can be deleted at any time through the admin interface

### Compliance

- SOC 2 Type II compliant
- GDPR compliant
- CCPA compliant

## Data deletion

You can delete your data at any time:

1. Navigate to Organization Settings
2. Select "Delete Organization Data"
3. Confirm deletion

All indexed content and metadata will be permanently removed within 24 hours.
